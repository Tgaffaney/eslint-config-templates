# Prettier config

## Usage

### Install

```bash
$ npm i @tylergaffaney/prettier-config -D
# OR
$ yarn add @tylergaffaney/prettier-config -D
```

### Extend this config

in `package.json`

```json
{
    "prettier": "@tylergaffaney/prettier-config"
}
```
